# pycharm-peojects
flask_web


基于flask与bootstrap框架搭建的博客系统。
仅支持python2.7版本。

<b>实现了用户的认证，关注，角色划分，邮件认证等功能。
支持markdown富文本编辑。</b>

<b>运行方式：</b>
下载源码，在flask_web文件夹使用命令行输入：python manage.py shell
导入设置。
输入app.run（）运行程序。
![](https://github.com/xianth123/pycharm-peojects/jietu.png)
用浏览器打开http://127.0.0.1:5000/即可进入博客系统。
